wordcloud
=========

WordCloud using t-SNE for Ersatz Labs

[Try it out!](http://rozifus.github.io/wordcloud/)

* Left Click:     Auto-Move To Word
* Right Click:    Rotate
* Middle Click:   Pan
* Scroll Wheel:   Zoom

* W: Move Forward
* S: Move Backward

* A: Move Left
* D: Move Right

* Q: Move Down
* E: Move Up
